Projeto Helpnei

Ver no GitHub

Desafio:

Os patrocinadores querem ver seus dados e a evolução deles de forma mais efetiva e fácil.

Solução:

Criar um site para o aplicativo da Helpnei, que mostre informações no dashboard e permita convites de patrocínio de usuários com base no perfil dos mesmos.

Desenvolvi um site que permite visualizar dashboard de cada empresa, realizar cadastro de usuários e pedir patrocínio de empresas interessadas no perfil do usuário cadastrado. Eu tive como responsabilidades integrar o Back com o Front, criar a API com CRUD dos dados do banco do site, criar o manual do usuário e realizar operações no git para juntar os commits.
